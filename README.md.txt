# Social-Media-Analytics-Tool-for-Twitter
It's end-to-end bi architecture for streaming data from Twitter then we clean it and apply sentiment analysis to detect the status of each tweet if it's positive, negative, or 
natural, and reply to each tweet based on its status, finally visualize the data

Technologies to be used
1- For streaming data from twitter use "Spark"
2- For queuing and persisting messages "Kafka"
3- For processing data from Kafka to Hive "Spark Streaming)"
4- For SQL on Hadoop "Hive"
5-For visualization "PowerBI"
